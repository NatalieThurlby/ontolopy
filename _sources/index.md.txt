# Ontolo**py**

```{image} images/ontolopy.png
```

<br>

Ontolopy is a Python package for manipulating biological ontologies. In particular, it is useful for linking sample metadata to ontologies, leveraging this link to get extra information about samples, grouping samples, etc. 

Ontolopy supports `.obo` files (it does not support `.owl` files).
